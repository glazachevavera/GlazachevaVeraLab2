### 1. Постановка задач

#### Задача 1

##### Условие задачи

> Ваша задача – посмотреть, какие значения принимает последовательность при разных a0 и n, вывести закономерность, по которой строится последовательность и запрограммировать ее самостоятельно.

#### Задача 2

##### Условиe задачи

> Дана последовательность из n чисел. Префиксом будем называть отрезок последовательности, который начинается с первого элемента и заканчивается на некоторой позиции. Найти первый префикс, на котором минимум и максимум имеют одинаковую четность, и вывести его длину (если такого префикса нет, вывести 0).

##### Пониманте условия

Минимально возможная длинна отрезка 2, так как, если 1, то задача не имеет смысла решения, так как для любой последовательности будет результат 1, так как максимальный и минимальный члены будут равны, как следствие их четности будут одинаковыми. Нам нужно найти первый отрезок, на котором максимум и минимум имеют одинаковую четность. Начальная длина отрезка на котором будем проверять выполнение условия 2, так как должно быть минимум два элемента для онализа.

#### Задача 3

##### Условие задачи

> Ввести с клавиатуры натуральное число N. Найти и вывести первые простые близнецы, которые больше числа N. Два простых числа называются близнецами, если разность между ними равна 2.

##### Пониманте условия

Нам нужно найти два простых числа, больших вводимого с клавиатуры числа N, различающиеся на 2.

#### Задача 4

##### Условие задачи

> Дан массив из n чисел. Найти минимальную разницу между любыми двумя элементами массива.

### 2. Входные и выходные данные

#### Задача 1

##### Входные данные

На вход программа получает целое число a0 (первый член последовательности, в коде программы обозначен как 'a', чтобы не вводить еще одну переменную (член последовательности)) и натуральное число n (от 1 до 99, требуемое количество следующих членов последовательности).

##### Выходные данные

На выход мы получаем последовательность из n чисел, в нашем случае из неотрицательных целых чисел (в алгоритме описано почему они тактие).

#### Задача 2

##### Входные данные

На  вход программа получает натуральное число n (количество членов последовательности) и n целых чисел (члены последовательности). 

##### Выходные данные

На выход мы получаем целое неотрицательное число (длинна отрезка, если не нашли нужного отрезка 0).

#### Задача 3

##### Входные данные

На вход программа получает натуральное число, обозначенное как 'N'.

##### Выходные данные

На выход мы получаем два натуральных числа.

#### Задача 4

##### Входные данные

На вход программа получает натуральное число (длина массива), обозначенное как 'n', а затем n вещественных чисел (элементы массива).

##### Выходные данные

На выход мы получаем неотрицательное число (0 если в массиве есть одинаковые числа).

### 3. Алгоритмы

#### Задача 1

1. **Ввод данных**
   
   Программа считывает целые числа (int), обозначенные как 'a' и 'n'.
   
2. **Вычисление следующего члена последовательности**

   Когда подставлялись разные значения a0 в условии к заданию, была выявлена закономерность: если среди цифр числа встречается 0, то последующие члены последовательности обнуляются. Было проверено перемножаются ли цифры члена последовательности для получения следующего. Также было замечено, что все выведенные члены последовательности делятся на 7. Так была получена закономерностть для каждого следующего члена последовательности. Каждый следующий член последовательности это произведение цифр числа предыдущего члена умнроженное на 7.
   
3. **Вывод результата**

   Программа выводит следующий член последовательности, если он не последний выводит запятую и пробел. Вычисление следующего члена последовательности и его вывод программа осуществляет заданное n раз.
   
#### Задача 2

1. **Ввод данных**
   
   Программа считывает целое число (int), обозначенные как 'n', затем еще n целых чисел.
   
2. **Нахождение первого отрезка, подходящего под условие, и вычисление его длины**

   За начальную длину отрезка берем 0, так как если отрезок не будет найден мы ее и выведем, не проверяя дополнительных условий. первые два члена обозначаем как максимальный и минимальный, в соответствии с их значениями относительно друг друга. Проверяем для них не составляют ли они нужный отрезок, если да выводим 2, если нет продолжаем искать нужный отрезок. Для каждого отрезка начиная с длины 3 сравневаем существующие минимальный и максимальный члены с новым введенным, если нужно заменяем максимальный или минимальный, сравниваем четность максимального и минимального членов. Если их четность одинаковая, то мы нашли нужный отрезок, если нет проверяем длинну на 1 большую. Повторяем действия, пока не проверим все возможные отрезки или не найдем нужный отрезок.
   
3. **Вывод результата**

   Программа выводит длинну первого, подходящего под условие отрезка, если такого отрезка не нашла выводит 0.

#### Задача 3

1. **Ввод данных**

   Программа считывает натуральное число (int), обозначенное как 'N'.
   
2. **Нахождение простого числа большего N, для которого число на 2 большее тоже простое**

   Начинаем с числа на 1 большего N, так как число N натуральное и любое простое число тоже, а нам нужно найти минимальное простое число большее N. Проверяем являются ли числа простыми взятое число и число на 2 большее его. Если число найдено выводим его и число на два большее, если нет увеличивоем проверяемое число на 1. Продолжаем выполнять действия, пока не найдем число удовлетворяющее условию.
   
3. **Вывод результата**

   Программа выводит найденное число и число большее его на 2.

#### Задача 4

1. **Ввод данных**
   
  Программа считывает целое число (int), обозначенные как 'n', затем еще n чисел (double), которые сохраняются в массив a[].
   
2. **Вычисление минимальной разницы между двумя элементами массива**

   Для каждого элемента массива вычисляем разницу между ним и каждым элементом массива, разницу с которым еще не сравнивали с существующей разницой. За начальную разницу берем разницу между первым и вторым элементами массива. Если находим разницу меньшую существующей, заменяем существующую на нее.
 
4. **Вывод результата**

   Программа выводит полученное число (double, минимальная разница между двумя элементами массива).
  
### 4. Программы

#### Задача 1
```java
import java.io.PrintStream;
import java.util.Scanner;
public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;
    public static void main(String[] args) {
        int a = in.nextInt();
        int n = in.nextInt();
        for (int i = 0; i < n; i++) {
            int p = 1;
            do {
                p *= a%10;
                a /= 10;
            } while (a != 0);
            a = p*7;
            out.print(a);
            if (i != n-1) {
                out.print(", ");
            }
        }
    }
}
```

```java
import java.io.PrintStream;
import java.util.Scanner;
public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;
    public static void main(String[] args) {
        int n = in.nextInt();
        int min = in.nextInt();
        int max = in.nextInt();
        if (max < min) {
            int a = max;
            max = min;
            min = a;
        }
        if ((max+min)%2 == 0) {
            out.print(2);
        }
        else {
            int l = 0;
            int k = 3;
            while (k <= n && l == 0) {
                int x = in.nextInt();
                if (x < min) {
                    min = x;
                }
                if (x > max) {
                    max = x;
                }
                if ((min+max)%2 == 0) {
                    l = k;
                }
                k++;
            }
            out.print(l);
        }
    }
}
```

```java
import java.io.PrintStream;
import java.util.Scanner;
public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;
    static boolean isSimple(int x) {
        int k = 0;
        for (int i = 2; i < Math.pow(x,0.5); i++) {
            if (x%i == 0) {
                k++;
            }
        }
        return k == 0;
    }
    public static void main(String[] args) {
        int N = in.nextInt();
        int x = N+1;
        while (!isSimple(x) || !isSimple(x+2)) {
            x++;
        }
        out.print(x);
        out.print(" ");
        out.print(x+2);
    }
}
```

#### Задача 4
```java
import java.io.PrintStream;
import java.util.Scanner;
public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;
    public static void main(String[] args) {
        int n = in.nextInt();
        double [] a = new double[n];
        for (int i = 0; i < n; i++)
            a[i] = in.nextDouble();
        double k = Math.abs(a[1] - a[0]);
        for (int i = 0; i < n-1; i++) {
            for (int j = i+1; j < n; j++) {
                if (Math.abs(a[i] - a[j]) < k) {
                    k = Math.abs(a[i] - a[j]);
                }
            }
        }
        out.print(k);
    }
}
```
